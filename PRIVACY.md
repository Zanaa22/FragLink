# Política de Privacidad de FragLink Última actualización: 13 de diciembre de 2025 
La extensión de Chrome FragLink está diseñada para ser una herramienta de búsqueda simple y privada. 
1. Recopilación de datos 
* No se recopilan datos personales: FragLink no recopila, almacena, transmite ni tiene acceso a ninguna información de identificación personal, historial de navegación, datos financieros o de salud de sus usuarios. 
* No se utiliza Google Analytics ni herramientas de seguimiento de terceros. 
2. Uso de la Información
 * La única información que maneja la extensión es el texto seleccionado por el usuario. Esta información se utiliza exclusivamente en el momento para construir la URL de búsqueda y abrir una nueva pestaña en el sitio web de Fragrantica. * Una vez que el usuario es redirigido a Fragrantica, el manejo de datos se rige por la política de privacidad de dicho sitio. 
3. Permisos 
* La extensión solo solicita el permiso contextMenus para agregar la opción de búsqueda al menú de clic derecho. 
4. Contacto: 
* Si tiene alguna pregunta, puede contactar con el desarrollador en fraglinkpro@gmail.com.
